## How to use Gulp.js?

1. Install node.js:

### For Windows
<http://nodejs.org/>

### For MacOS

>$ brew install node

2. Install gulp globally:

>$ npm install --global gulp

3. Install gulp in your project devDependencies:

>$ npm install --save-dev gulp

4. Install gulp plugins

>$ npm install

5. Run gulps:

>$ gulp
